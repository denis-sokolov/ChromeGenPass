Instructions

  1. Store master password on this options page.

    Don’t worry, the password itself is not stored, only its hash is.

  2. On any website, focus a password box and choose your password from the menu by pressing buttons 1 to 9.

  3. The first time you use a password since opening your browser, the extension will ask you to unlock it.

    Careful! Evil website owners might want to steal your password, so always make sure that the popup says ‘SuperGenPass for Google Chrome™ by Denis extension says’ and not ‘The page at ... says:’!

  4. That’s it, the password field is now filled with your generated password.

Whitelist

If you do not want for this extension to run on some domains (say, you have your own password there), you can whitelist that domain in this field.
